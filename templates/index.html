Control

This section allows you to change between the different channels and set the main configuration:

  • Channel hopper: if enabled, the channel will be automatically changed
  • Change to optimal channel: change to the current optimal channel manually
  • Change to channel: change to desired channel among available
  • Current channel: channel being used
  • Optimal channel: current optimal channel

    • Devices

  • Table: contains data about all devices in the environment
  • Devices graph: shows the amount of devices connected to each channel
  • Bytes graph: shows the amount of bytes transmitted in each channel recently
